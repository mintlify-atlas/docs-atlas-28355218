## Overview

The `Armors` class manages all armor sets in the Elemental Wizards mod, providing methods for creating custom armor materials and registering complete armor sets with spell power bonuses.

## Class: Armors

**Package:** `net.elemental_wizards_rpg.item.armor`

### Methods

#### material()

Creates and registers a custom armor material with specified protection values and properties.

```java
public static RegistryEntry<ArmorMaterial> material(
    String name,
    int protectionHead, 
    int protectionChest, 
    int protectionLegs, 
    int protectionFeet,
    int enchantability, 
    RegistryEntry<SoundEvent> equipSound, 
    Supplier<Ingredient> repairIngredient
)
```java

**Parameters:**
- `name` - Identifier name for the armor material
- `protectionHead` - Protection value for helmet
- `protectionChest` - Protection value for chestplate
- `protectionLegs` - Protection value for leggings
- `protectionFeet` - Protection value for boots
- `enchantability` - Enchantment value (higher = better enchantments)
- `equipSound` - Sound event when equipping armor
- `repairIngredient` - Ingredient supplier for repairing the armor

**Returns:** `RegistryEntry<ArmorMaterial>` - The registered armor material

**Example:**
```java
public static RegistryEntry<ArmorMaterial> material_elemental = material(
    "elemental",
    1, 3, 2, 1,
    9,
    SoundEvents.ITEM_ARMOR_EQUIP_LEATHER, 
    WOOL_INGREDIENTS
);
```java

#### register()

Registers all armor sets with their configurations.

```java
public static void register(Map<String, ArmorSetConfig> configs)
```java

**Parameters:**
- `configs` - Map of armor set configurations

### Elemental Armor (Tier 1)

**ID:** `elemental_wizards_rpg:elemental`

**Bonuses per piece:**
- Air Spell Power: +20%
- Earth Spell Power: +20%
- Water Spell Power: +20%
- Fire Spell Power: +20%

**Example:**
```java
public static final Armor.Set elementalArmor = create(
    material_elemental,
    Identifier.of(MOD_ID, "elemental"),
    10,
    ElementalRobe::new,
    ArmorSetConfig.with(
        new ArmorSetConfig.Piece(1)
            .addAll(List.of(
                AttributeModifier.multiply(MoreSpellSchools.AIR.id, 0.2F),
                AttributeModifier.multiply(MoreSpellSchools.EARTH.id, 0.2F),
                AttributeModifier.multiply(MoreSpellSchools.WATER.id, 0.2F),
                AttributeModifier.multiply(SpellSchools.FIRE.id, 0.2F)
            )),
        // ... other pieces
    ), 1, null
).armorSet();
```java

## Creating Custom Armor Sets

To create a custom armor set:

1. Define an armor material using `material()`
2. Create armor set configuration with `ArmorSetConfig.with()`
3. Add attribute modifiers for each piece
4. Register using the `create()` method

```java
public static final Armor.Set customArmor = create(
    customMaterial,
    Identifier.of(MOD_ID, "custom"),
    20,  // durability multiplier
    ElementalRobe::new,
    ArmorSetConfig.with(
        new ArmorSetConfig.Piece(1)  // helmet
            .add(AttributeModifier.multiply(SpellSchools.FIRE.id, 0.3F)),
        new ArmorSetConfig.Piece(3)  // chestplate
            .add(AttributeModifier.multiply(SpellSchools.FIRE.id, 0.3F)),
        new ArmorSetConfig.Piece(2)  // leggings
            .add(AttributeModifier.multiply(SpellSchools.FIRE.id, 0.3F)),
        new ArmorSetConfig.Piece(1)  // boots
            .add(AttributeModifier.multiply(SpellSchools.FIRE.id, 0.3F))
    ),
    2,  // tier
    null
).armorSet();
```java
