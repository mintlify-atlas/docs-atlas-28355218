## Overview

The `WeaponsRegister` class manages all magical weapons (wands and staffs) in the Elemental Wizards mod, handling weapon registration, materials, and spell power attributes.

## Class: WeaponsRegister

**Package:** `net.elemental_wizards_rpg.item.weapons`

### Fields

```java
public static final ArrayList<Weapon.Entry> entries = new ArrayList<>();
```java

Stores all registered weapon entries.

### Methods

#### entry()

Creates a weapon entry with the specified properties.

```java
private static Weapon.Entry entry(
    String name, 
    Weapon.CustomMaterial material, 
    Weapon.Factory factory, 
    WeaponConfig defaults, 
    Equipment.WeaponType category
)
```java

**Parameters:**
- `name` - Weapon identifier name
- `material` - Custom material for the weapon
- `factory` - Factory for creating weapon items
- `defaults` - Default weapon configuration (damage, speed)
- `category` - Weapon category type

**Returns:** `Weapon.Entry` - The created weapon entry

#### ingredient()

Creates an ingredient supplier that handles mod dependencies.

```java
private static Supplier<Ingredient> ingredient(
    String idString, 
    boolean requirement, 
    Item fallback
)
```java

**Parameters:**
- `idString` - Item identifier string
- `requirement` - Whether the mod is loaded
- `fallback` - Fallback item if mod is not loaded

**Returns:** `Supplier<Ingredient>` - Ingredient supplier

#### wand()

Creates a wand weapon entry with standard wand properties.

```java
private static Weapon.Entry wand(String name, Weapon.CustomMaterial material)
```java

**Parameters:**
- `name` - Wand identifier name
- `material` - Custom material for the wand

**Returns:** `Weapon.Entry` - The created wand entry

#### staff()

Creates a staff weapon entry with standard staff properties.

```java
private static Weapon.Entry staff(String name, Weapon.CustomMaterial material)
```java

**Parameters:**
- `name` - Staff identifier name
- `material` - Custom material for the staff

**Returns:** `Weapon.Entry` - The created staff entry

#### register()

Registers all weapons with their configurations.

```java
public static void register(Map<String,WeaponConfig> configs)
```java

**Parameters:**
- `configs` - Map of weapon configurations

```java

    Weapon.CustomMaterial.matching(ToolMaterials.NETHERITE, () -> Ingredient.ofItems(Items.IRON_BLOCK)))
    .attribute(AttributeModifier.bonus(MoreSpellSchools.EARTH.id, T4_STAFF_POWER))

    .attribute(AttributeModifier.bonus(MoreSpellSchools.AIR.id, T4_STAFF_POWER))

    .rarity = Rarity.RARE;
```java
- Earth/Water/Air Spell Power: +8 each
- Grants: Elemental Avatar spell
- Rarity: RARE
- Requires: Arsenal mod

## Creating Custom Weapons

### Custom Wand Example

```java
public static final Weapon.Entry customWand = wand("custom_wand",
    Weapon.CustomMaterial.matching(ToolMaterials.DIAMOND, () -> Ingredient.ofItems(Items.DIAMOND)))
    .attribute(AttributeModifier.bonus(SpellSchools.FIRE.id, 6F))
    .loot(Equipment.LootProperties.of(2));
```java

### Custom Staff Example

```java
public static final Weapon.Entry customStaff = staff("custom_staff",
    Weapon.CustomMaterial.matching(ToolMaterials.NETHERITE, () -> Ingredient.ofItems(Items.NETHERITE_INGOT)))
    .attribute(AttributeModifier.bonus(MoreSpellSchools.WATER.id, 8F))
    .attribute(AttributeModifier.multiply(SpellPowerMechanics.HASTE.id, 0.1F))
    .loot(Equipment.LootProperties.of(3));
```java
